# Calculator App (Python)

A simple calculator application built using Python that performs basic arithmetic operations based on user input.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Menu-based user interaction

## How to Run
1. Make sure Python is installed
2. Run the file using:
   python calculator.py

## Concepts Used
- Python basics
- Conditional statements
- Functions
- User input handling
